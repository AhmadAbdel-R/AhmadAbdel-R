# Ahmad Abdel-Raheem

I am an Electrical Engineering student building embedded systems that combine custom hardware and firmware.

Most of my work is hands-on integration: PCB design, MCU bring-up, telemetry links, and system-level debugging.

## Main Projects

- `Capstone-Drone-Remote-Firmware`
  - ESP32-S3 remote firmware, dual nRF links, display UI
- `Drone_remote_hardware`
  - KiCad designs for remote/FCU boards and nRF breakout work
- `easy-esc-esp32`
  - ESP32 DShot/RMT ESC control library
- `SPI_to_CAN`
  - ESP32 + MCP2515 CAN bridge with PWM-based sensing
- `NRF2401-Template`
  - practical transmitter/receiver templates for nRF24 links

## Tools I Use

- C/C++ (Arduino + ESP-style embedded workflows)
- PlatformIO / VS Code
- KiCad
- logic analyzer / oscilloscope based bring-up
